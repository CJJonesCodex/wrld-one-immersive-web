# Phase 5.6 — Text Monument VFX Worlds

## 1) Why this phase exists
- Test text-first, interaction-led world moments inspired by PRMO quality principles.
- Reduce reliance on image/video cards as the primary hero moment.
- Strengthen typographic identity by turning titles into cinematic objects.
- Give each world a distinct procedural VFX signature.

## 2) Visual rule
- One active world title at a time.
- One title VFX layer at a time.
- No inactive title labels.
- No duplicate View bubbles.
- No wall of cards.
- No picture-first layout in this mode.

## 3) Visual mode switch
- `title-vfx` mode is the default.
- `media-cards` mode is preserved and can be restored later.

## 4) World VFX identities
- Living Macro: organic dew shimmer.
- Signal Garden: signal lines + node pulses.
- Core Chamber: concentric halo system.
- Aurora Passage: ribbon refraction field.
- Rift Bloom: petal aperture bloom.
- Future World: constellation blueprint field.

## 5) Build verification
- `npm run build` must pass.
