# Phase 4 Reference Brief — PRMO-Inspired Bright Spatial System (WRLD ONE)

## 1) Observed interaction principles from PRMO (`prm.ooo`)

Reference reviewed: https://www.prm.ooo/

- **Featured Works as a core narrative rail** with numbered entries that remain visible while exploring.
- **Numbered project list** creates editorial rhythm and fast spatial orientation.
- **Scroll-to-explore behavior** feels like moving through an interactive world, not reading stacked sections.
- **Sound / Haptic / 3D Sensor controls** are surfaced as explicit sensory toggles.
- **3D-per-section feeling** gives each segment a different atmosphere and depth identity.
- **3D slider feeling** allows project focus to shift smoothly with depth and motion, not abrupt page swaps.
- **Custom cursor feeling** communicates system state and interaction intent.
- **Audio visualizer feeling** acts as a small ambient system indicator when sound is enabled.
- **Scrollbar-to-navigation feeling** maps progress and allows direct jumps to specific works.
- **Premium smooth motion language** uses damping and elegant interpolation to feel cinematic.

## 2) WRLD ONE adaptation strategy

- Convert **Featured Works → Featured Worlds**.
- Convert flat project cards into **floating 3D world/media capsules**.
- Convert PRMO-like sensory toggles into **WRLD ONE Sensory Controls** (sound, haptic, sensor).
- Convert generic scroll into a **cinematic camera journey through spatial media**.
- Convert the previous dark/minimal foundation into a **bright, optimistic, premium spatial interface**.

## 3) What must not be copied

- PRMO brand identity.
- PRMO logo.
- PRMO exact layout.
- PRMO exact text/copywriting.
- PRMO project names.
- PRMO assets/media.
- PRMO source code.
